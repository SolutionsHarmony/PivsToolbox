# Security Policy

## Reporting a vulnerability

If you discover a security or privacy issue in any tool in this repository,
please report it privately rather than opening a public issue.

- **Email:** contact@solutionsharmony.com
- Please include steps to reproduce and the affected tool/version.

We aim to acknowledge reports within a few business days and will keep you
updated as we investigate and resolve the issue.

## Scope

This repository contains small client-side tools (including the Token Monster
Chrome extension). The extension reads only your own Claude usage in your own
logged-in browser — see [`ClaudeTime/PRIVACY.md`](ClaudeTime/PRIVACY.md).
