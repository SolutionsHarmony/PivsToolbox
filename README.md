# Piv's Toolbox

A small collection of free tools and utilities by **[Solutions Harmony](https://solutionsharmony.com)**.

Each tool lives in its own folder and can be used independently. Everything here is
free for **noncommercial** use (see [License](#license)).

## What's inside

### 🍪 Token Monster — Claude usage tracker (Chrome extension)
`ClaudeTime/`

A Chrome extension that shows your real Claude 5-hour and weekly usage as a fun,
at-a-glance progression — driven by the usage you're already signed in to on
`claude.ai`. No API keys, no separate login, no passwords; it reads only your own
usage in your own logged-in browser. See [`ClaudeTime/README.md`](ClaudeTime/README.md)
and [`ClaudeTime/PRIVACY.md`](ClaudeTime/PRIVACY.md).

> **Not affiliated with, endorsed by, or sponsored by Anthropic or Sesame Workshop.**

### 🗂️ AppCard — operational handbook skill + tooling
`skills/appcard/`, `scripts/`, `installer/`

An [Claude Code](https://claude.com/claude-code) skill (and supporting install
scripts) for generating and maintaining an `appcard.md` operational handbook for a
repository. See [`skills/appcard/SKILL.md`](skills/appcard/SKILL.md).
